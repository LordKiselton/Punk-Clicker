# Разбор UI референса (Tappers of Grayskull / архетип Tap Titans) → «Балаган»

> Источник: Tappers = клон Tap Titans; ниже — детальная раскладка архетипа + подтверждённые
> особенности Tappers, и что из этого берём. Публичных скринов Tappers мало — если нужна
> пиксель-точность, могу открыть реальные скрины через браузер (нужно подтверждение).

## Общая архитектура экрана (портрет)

Экран делится на две зоны: **верх ~60–65% — бой**, **низ ~35% — прокачка/меню**.
Между ними — **нижний таб-бар** навигации. Это каркас всего жанра.

```
┌─────────────────────────────┐
│ золото ▾/сек    [гемы][прест]│  ← верхняя строка валют
│        Стадия 142   ●●●●○○○○ │  ← № стадии + «пипсы» волны (убито 4/8)
│                              │
│        [ФОН ЛОКАЦИИ]         │
│            ВРАГ/БОСС         │  ← тап по верхней зоне = удар
│   герой ◄  ► союзники(фланг) │
│                       [навык]│  ← иконки активных навыков (радиальный КД)
│   🧚 бонус-фея (реклама)      │
├─────────────────────────────┤
│  [контент активной вкладки]  │  ← напр. список героев
│  [x1 / x10 / x100 / MAX]     │  ← множитель покупки (важно!)
├─────────────────────────────┤
│ Мастер Герои Навыки Снаряж … │  ← нижняя навигация (иконки+подписи)
└─────────────────────────────┘
```

## Боевая зона — детали

- **Тап по верхней области = удар** (в Tappers ещё «свайп — обычная атака, тап — по слабому месту»). У нас оставляем просто тап.
- **Враг** — крупно по центру/чуть правее, перед ним **герой слева**, **союзники выстроены по нижнему краю флангом** и бьют автоматически с мини-анимациями. Это ключевой приём: видно «армию», а не абстрактный DPS.
- **Числа урона** разлетаются: белые — обычные, жёлто-оранжевые крупные — криты/навыки. (Мы уже так делаем + цвет на каждого героя — это даже лучше референса.)
- **Дроп**: монеты «выпадают» с убитого и летят к счётчику.
- **Бонус-фея** (в Tappers — Орко): иконка изредка пролетает по экрану, тап даёт бесплатный бонус или «посмотри ролик за золото/гемы». → у нас это идеальный слот для **rewarded** и для **Болтливой Головы** как талисмана.

## Индикатор прогресса (берём обязательно)

- **Номер стадии** + ряд **пипсов волны** (напр. ●●●●○○○○ — убито 4 из 8 на стадии).
- **Босс**: вместо пипсов — надпись BOSS и **красная полоса таймера**, утекающая за 30 сек. (Сейчас у нас текст-таймер — заменим на полосу.)

## Панель героев (Heroes) — эталон

Вертикальный **скролл-список**, каждая строка:
- круглый **портрет** героя,
- **имя** + текущий **уровень**,
- его **DPS/вклад**,
- крупная кнопка **«Улучшить»** с ценой (гаснет, если не хватает золота),
- маркеры **вех** (milestone) на ключевых уровнях (×бонус),
- иногда «эволюция»/разблок навыка союзника.
Сверху списка — **переключатель множителя покупки x1 / x10 / x100 / MAX** (огромный QoL, must-have).

## Прочие экраны (вкладки нижней навигации)

- **Мастер меча** (у нас «Шут») — прокачка урона за тап, личные навыки героя.
- **Герои** (у нас «Труппа») — см. выше.
- **Навыки** — активные способности с деревом/прокачкой.
- **Снаряжение / Питомцы / Артефакты** — у нас можно свести к **реликвиям** (drop с боссов) и **мета-дереву prestige** («Сказка»).
- **Магазин** (у нас «Лавка») — гемы/черепа, IAP, «убрать рекламу», офферы.
- **Prestige** — отдельное подтверждение со счётчиком метавалюты.

## Арт-стиль референса

Tappers — яркий **мультяшный** стиль, верный мультсериалу 80-х: насыщенные цвета, толстый контур, комикс. UI — **фэнтезийные рамки** (камень/дерево, золотая окантовка), «пергаментные» панели, светящиеся кнопки.

**Что берём, что меняем под себя:**
- ✅ Берём: композицию (бой/меню/таб-бар), пипсы волны, полосу-таймер босса, список героев с портретами и множителем покупки, ряд навыков с радиальным КД, бонус-фею под rewarded, союзников-флангом в сцене, орнаментальные рамки.
- 🔄 Меняем: **палитру и тон** — не яркий мультик, а наша **тёмная панк-сказка** (баклажан/кровь/золото). Рамки — не «золотое фэнтези», а **дерево/кость/балаган**. Это наша визуальная подпись.

## Что обновить в нашем UI (приоритет)

1. **Пипсы волны** + **полоса таймера босса** вместо текста.
2. **Список Труппы**: портреты-кружки + DPS + кнопка-цена + **множитель x1/x10/x100/MAX**.
3. **Союзники в сцене** флангом у Шута (видимая «труппа»), а не только в списке.
4. **Ряд навыков** с радиальным кулдауном (когда добавим активки).
5. **Бонус-фея/Голова** пролетает с rewarded-оффером.
6. Орнаментальные рамки панелей (дерево/кость) вместо плоских.
