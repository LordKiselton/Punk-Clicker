## ✅ Что уже готово

**Технический фундамент**
- Godot 4.6.1, сборка/подпись Android APK, установка на устройство, GitHub-репо + ветки.
- Тулчейн (JDK17, Android SDK), скрин-луп для итерации UI без сборки.

**Кор-геймплей**
- Петля: тап → золото → прокачка удара + найм героев (idle-DPS) → стадии → боссы с таймером.
- Экономика: формулы + симулятор баланса, отбалансирован первый проход.
- Сейв/загрузка + базовый оффлайн-доход.

**Контент (текущая сборка)**
- 3 героя (Рыцарь/Ведьма/Шут) с артом + закрытый ростер «Скоро».
- 2 врага (Зомби/Вервольф) с чередованием, фон-локация (лес).

**UI / Feel**
- Редактируемая сцена «Сцена труппы»: рельс карточек-героев, панель действий, валюты.
- HP-бар со «следом урона», вылет цифр из врага, вспышка/тряска, смерть врага, полёт монет.
- Множитель покупки x1/x10/x100/MAX, свайп-скролл, неактивные кнопки при нехватке золота.

**Монетизация (де-риск)**
- Yandex rewarded (демо) работает на устройстве; пауза игры на время ролика.
- Плагин RuStore Billing найден (официальный) — ещё не интегрирован.

## 🎯 Что должно быть в хорошем кликере на релизе

1. **Глубокая прогрессия:** prestige-сброс с метавалютой и мета-деревом; много героев; активные навыки; вехи/эволюции; несколько локаций и боссов.
2. **Ретеншн-петли:** оффлайн-доход с экраном возврата, ежедневные награды, квесты/достижения, пуш-уведомления.
3. **Монетизация (по-настоящему):** rewarded (реальные ID), IAP (убрать рекламу, паки валюты, перм-бусты, стартовый оффер), аккуратные interstitial.
4. **Полировка/feel:** звук и музыка, онбординг-туториал, экран настроек, переходы, иконка/сплеш.
5. **Качество/тех:** корректные большие числа, локализация (RU), устойчивый сейв, аналитика, производительность.
6. **Готовность к стору:** аккаунт RuStore, карточка (иконка/скрины/описание/возрастной рейтинг/политика конфиденциальности), релизный AAB.

## 📦 Минимум для первого релиза (lean v1.0)

Не обязательно делать всё сразу — часть контента растим уже после запуска (live-обновления). Минимально «не стыдно выложить»:

1. Кор-петля + **prestige** + **оффлайн-экран** (Фаза 1, 🔴).
2. Базовый **арт** ключевых экранов + **звук/музыка** + **туториал** (Фазы 2–3 частично).
3. **Реальный rewarded** + **«убрать рекламу» (IAP)** как минимум (Фаза 4, 🔴).
4. **Стор-готовность**: AAB, карточка, политика, рейтинг (Фаза 6, 🔴).

Остальное (полный ростер, навыки, дейлики, аналитика, локации) — пострелизными апдейтами.
