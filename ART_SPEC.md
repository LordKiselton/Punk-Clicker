# Тех-требования к арту «Балагана»

> Для генерации/рисовки ассетов под текущий UI ([VISION.md](VISION.md) · [LORE.md](LORE.md)).
> Пока в игре — векторные заглушки; этот документ задаёт формат финального арта, чтобы он встал без переделок.

## Базовые параметры

- **Дизайн-разрешение игры:** 720×1280 (портрет), растяжение `canvas_items`. Рисуем арт в **2×** под современные экраны (≈1080p+).
- **Движок:** Godot 4.6.1. Импорт PNG/WebP «из коробки», VRAM-компрессия ETC2/ASTC включена.
- **Прозрачность:** все персонажи, враги, иконки, рамки — **PNG-32 с альфа-каналом** (прозрачный фон). Фоны локаций — без альфы (PNG или WebP q85).
- **Лимит размера текстуры:** ≤ 2048 px по большей стороне (совместимость). Бюджет арта в APK: ориентир **до ~50 МБ** суммарно (фоны — в WebP).

## Список ассетов

### Персонажи и враги (PNG-32, прозрачный фон, в полный рост, по центру кадра)

| Ассет | Размер (px) | Ориентация | Заметки |
|---|---|---|---|
| Шут (герой) | ~500×700 | смотрит **вправо** | поза idle + поза удара (2 кадра) для сочности тапа |
| Союзник ×4 (Голова, Громобой, Ведьма, Путник) | ~300×450 | смотрит **вправо** | idle; стоят слева рядом с Шутом |
| Рядовой враг ×4 на локацию | ~400×400 | смотрит **влево** | idle; по 4 типа на каждую из 6 локаций |
| Босс ×1 на локацию | ~640×640 | смотрит **влево** | крупнее, читаемый силуэт |

> Якорь: ступни внизу кадра по центру по X. Единое направление света (сверху-слева). Толстый контур, плоская заливка — стиль «тёмная книжка-сказка».

### Фоны локаций (6 шт., 1080×1920, без альфы)

Лес · Погост · Трактир · Усадьба · Ярмарка · Замок (см. [LORE.md](LORE.md)).
Композиция: **верхние ~55%** — сцена/атмосфера; **средняя «арена-полоса»** — где стоит враг (не загромождать центр); **нижние ~35%** — притемнить/упростить (поверх ляжет UI-панель и таб-бар, детали не нужны).

## Подсказки по AI-генерации

- **Единый стиль:** один «стилевой якорь» (референс-кадр) + фиксированный seed, чтобы герои/враги были в одной вселенной.
- **Прозрачность:** многие генераторы не дают альфу — генерь на сплошном контрастном фоне (напр. чистый `#00ff00`), затем вырезай фон; на выходе — PNG с альфа.
- **Промпт-скелет (персонаж):** «full-body character, centered, facing right, transparent/flat background, flat 2D storybook illustration, dark punk fairy tale, thick ink outline, limited palette (#1a1320 #b5121b #e0c341 #6f9f5a), grim but playful, Russian dark folklore vibe — [кто именно, из LORE.md]».
- **Враги:** то же, но `facing left`, `monster/creature`.
- **Фоны:** `vertical 9:16 background, [локация], atmospheric, darker bottom third for UI, flat 2D storybook, same palette`.
- Согласуем по одному типу за раз, я встрою и проверю на телефоне.

## Как встраиваем

Кладёшь файлы по путям выше → я заменяю векторные заглушки на `TextureRect`/`Sprite2D` в тех же слотах (UI уже спроектирован под эти размеры) и собираю APK. Размеры держим как в таблице — тогда ничего не «поедет».
