# ГДД — «Балаган»

Боевой кликер (панк-сказка) для RuStore. Движок: Godot 4 / GDScript. Платформа: Android, портрет.
Связанные документы: [VISION.md](VISION.md).

---

## 1. Кор-петля (Core Loop)

```
ТАП по нечисти ──► наносим урон ──► враг умирает ──► падает ЗОЛОТО
       ▲                                                    │
       │                                                    ▼
   смотрим, как растут числа        тратим ЗОЛОТО на прокачку:
       │                            • +урон Шута за тап
       │                            • найм/прокачка соратников (idle DPS)
       │                            • навыки и бусты
       │                                                    │
       └──────────── сильнее ◄── продвигаемся по ТРОПЕ ◄────┘
                                  (стадии → боссы → новые локации)

Периодически: PRESTIGE («Новая сказка») ──► сброс прогресса
              за метавалюту «бубенцы» ──► следующий заход быстрее и дальше
```

**Микро-петля (секунды):** тап → урон → золото → апгрейд.
**Средняя петля (минуты):** добить стадию → босс (таймер!) → новая локация.
**Мета-петля (часы/дни):** накопить, упереться в стену → prestige → пробить дальше.

---

## 2. Валюты

| Валюта | Тип | Источник | Тратится на |
|---|---|---|---|
| **Золото** | мягкая | дроп с врагов, оффлайн-доход, rewarded | урон за тап, найм/левелап соратников, навыки |
| **Бубенцы** | мета (prestige) | при prestige (зависит от макс. стадии) | перманентные мета-апгрейды (множители) |
| **Черепа** | премиум (hard) | IAP, изредка квесты/реклама | ускорители, мгновенный найм, пропуск, косметика |

## 3. Боевая система и стадии

- **Стадия** = N врагов подряд (по умолчанию **10**), последняя — мини-волна, каждые **5 стадий — БОСС**.
- **Босс** = таймер **30 сек**: не успел убить → откат на 1 стадию (не дальше начала локации). Создаёт «стену», мотив прокачки и rewarded.
- **Локация** = блок из ~50 стадий со своим фоном/врагами/музыкой (Лес → Погост → Трактир → Ярмарка → Замок → …). Зацикливаются с реролом палитры после последней.
- **Тап:** урон = `tap_damage`. Враг перед Шутом получает урон; крит-шанс (база 5%, x2 урона).
- **Idle:** суммарный `dps` соратников бьёт врага автоматически каждую секунду (и оффлайн).

### Формулы (числовая основа — это «движок» экономики)

Стандарт жанра — экспоненты. Параметры подбираем при балансе, ниже — стартовые.

**HP врага на стадии s:**
`hp(s) = 10 * 1.55^(s-1)` (боссы ×10 к HP, ×8 к золоту)

**Золото с врага на стадии s:**
`gold(s) = 1 * 1.5^(s-1)`

**Урон за тап (уровень L прокачки):**
`tap_damage = base_tap * (1.08^L) * prestige_mult * boost_mult`

**Соратник i, DPS на уровне Lᵢ:**
`dps_i = base_i * Lᵢ * (1.1^floor(Lᵢ/25))` (каждые 25 ур. — milestone-бонус ×, как в Tap Titans)
`total_dps = Σ dps_i * prestige_mult * boost_mult`

**Стоимость уровня (любой апгрейд):**
`cost(L) = base_cost * growth^L`, где growth ≈ 1.07–1.12 (у разных сущностей разный).

> Принцип баланса: рост `gold(s)` ≈ рост `cost(L)`, чтобы каждые ~N минут игрок мог взять следующий значимый апгрейд, но упирался в стены перед боссами (→ повод для rewarded/prestige).

## 4. Соратники (труппа) — idle-ядро

~15–20 фольклорных архетипов, открываются за золото по мере тропы. Каждый:
- свой `base_dps`, своя `cost growth`, своя кривая;
- **milestone-бонусы** каждые 10/25/50/100 уровней (×DPS, разблок навыка, баф всей труппе);
- визуально появляется на сцене рядом с Шутом.

Стартовый список (рабочий): Лесник, Ведьма, Кукольник, Могильщик, Слепой гусляр, Палач, Юродивый, Шарманщик, Знахарка, Король-в-изгнании, Плакальщица, Костяной пёс, Болотница, Звонарь, Главарь шайки. *(имена-нарицательные = фольклор, без реального IP)*

---

## 5. Навыки (активные способности Шута)

5 активных навыков на кулдауне (тап по иконке), классика жанра:
- **Раж** — +урон тапа на 10 сек.
- **Барабанный бой** — +DPS труппы на 20 сек.
- **Золотой час** — ×N золота на 30 сек (важен для rewarded-синергии).
- **Шквал** — авто-тапы за игрока (10 сек).
- **Гроза** — мгновенный массовый урон.
Прокачиваются за золото (длительность/сила/кулдаун).

---

## 6. Prestige — «Новая сказка»

- Доступен после стадии **75** (первый раз — обучающий).
- Сброс: золото, уровни тапа, соратники, навыки, стадия.
- Сохраняется: **бубенцы**, мета-апгрейды, черепа, реликвии.
- **Награда бубенцами:** `bells = floor( k * (maxStage)^1.5 )` — резкий рост от рекорда → выгодно заходить дальше.
- **Мета-дерево (за бубенцы):** перманентные `prestige_mult` к урону/DPS/золоту, +стартовое золото, +оффлайн-кап, доп. слоты реликвий.
- **Реликвии** (опц., 2-я итерация): редкие пассивки, дроп с боссов.

---

## 7. Оффлайн-доход и возврат

- Накопление `total_dps`-эквивалента золота в офлайне, **кап 8 часов** (расширяется за черепа/мета).
- При входе: экран «Пока тебя не было…» с суммой + кнопка **«Удвоить (реклама)»**.
- Пуш-уведомление «балаган заскучал» через X часов (вернуть игрока).

---

## 8. Монетизация (точки интеграции)

**Rewarded-видео (Yandex) — основной доход, всё опционально и в плюс игроку:**
- Удвоить оффлайн-доход.
- «Золотой час» ×N на 30–60 мин.
- Сундук черепов/золота (кулдаун).
- Мгновенно добить босса при провале таймера (1 раз/бой).
- Ускорить кулдаун навыков.

**Interstitial:** редко и аккуратно — напр. при prestige (не чаще раза в N минут), чтобы не бесить.

**IAP (RuStore Billing):**
- «Убрать баннер» (разовая).
- Паки черепов.
- Перманентные бусты ×2 урон / ×2 золото (главные «киты»-покупки).
- Стартовый пак новичка (оффер первой сессии).
- Косметика труппы/Шута (скины, цвет балагана).

**Баннер:** опц., снизу, отключается покупкой. Решим — возможно вовсе без баннера ради ощущения качества.

## 9. UI / Экраны

**Главный боевой экран (одно касание до всего):**
```
┌──────────────────────────┐
│ [золото] [бубенцы] [череп]│  ← топ-бар валют
│        ЛОКАЦИЯ · стадия N │
│                          │
│      [враг / БОСС]        │  ← тап-зона (весь центр)
│   труппа ◄ ШУТ            │
│                          │
│ [навыки: 5 иконок CD]     │
├──────────────────────────┤
│ [Шут] [Труппа] [Навыки]   │  ← нижние табы-панели (выезжают)
│ [Тропа] [Магазин] [Сказка]│
└──────────────────────────┘
       [баннер опц.]
```
- **Шут:** прокачка урона за тап.
- **Труппа:** список соратников, найм/левелап, milestones.
- **Навыки:** прокачка активок.
- **Тропа:** прогресс по локациям, телепорт на пройденные.
- **Магазин:** черепа, IAP, rewarded-офферы, убрать рекламу.
- **Сказка (prestige):** мета-дерево, кнопка «Новая сказка».
- Поверх: оффлайн-экран, туториал, поп-апы офферов.

## 10. Сохранения

- Локально `user://save.json`: валюты, уровни, стадия/рекорд, соратники, навыки, мета-дерево, таймстемпы (для оффлайна), флаги покупок.
- Автосейв по таймеру + при сворачивании. Защита от отмотки времени (sanity-cap).
- Облако — позже (опц.).

## 11. Арт / Аудио (направление)

- **Арт:** рисованный 2D, мрачно-мультяшный, силуэтно-читаемый под маленький экран. Палитра: чёрный/бордовый/болотно-зелёный + акцентный «балаганный» жёлтый.
- **Аудио:** ударный фолк-панк луп на локацию (барабан/гусли/гармонь с панк-драйвом), сочные SFX удара/смерти/монет (game-feel критичен для кликера).
- **Juice:** тряска экрана на боссе, всплеск монет, всплывающие числа урона, crit-вспышка, бубенцы при тапе. Это половина успеха кликера.

## 12. MVP (что делаем в первую очередь) vs Полная версия

**MVP (играбельный прототип → первая сборка):**
1. Тап-бой по 1 врагу, золото, HP/gold-формулы.
2. Прокачка урона за тап.
3. 3–4 соратника (idle DPS) с наймом/левелапом.
4. Стадии + боссы с таймером, 1–2 локации (плейсхолдер-арт).
5. Сохранение + базовый оффлайн-доход.
6. **Заглушки монетизации** (rewarded/IAP вызываются, логируются) — для де-риска плагинов.
7. Сборка APK, запуск на устройстве.

**Дальше:** все соратники, навыки, prestige + мета-дерево, реальные Yandex/RuStore SDK, баланс, арт, звук, туториал, офферы, иконка, карточка RuStore, релиз.

## 13. Открытые числовые параметры (подберём при балансе)

`base_tap`, `base_dps` соратников, точные `growth` по каждой сущности, `k` в формуле бубенцов, HP/gold множители боссов, кап оффлайна. Сначала заложим формулы с параметрами в конфиг (легко крутить), потом отбалансируем на плей-тестах.
